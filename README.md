# YelpCamp
YelpCamp is a yelp-like website for users to post information about campgrounds and leave comments about exsiting campgrounds.

## Intended Users
People who love camping. 

## Technology Stacks
### Backend
1. Express.js
2. Node.js
3. Passport.js
4. MongoDB

### Frontend
1. CSS
2. HTML
3. JavaScript
4. Bootstrap
